SimplyReliableInfra class

One of the core and the most important classes of the API. This class is responsible for setting up the infrastructure and initialization required by the API. Some of the common functions of this class is to help the API user to build new messages, sending messages, setting the custom HTTP headers, sending infrastructure lifecycle events to all, to name a few.
