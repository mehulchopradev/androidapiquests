Message class

This class at the base represents an abstraction representation of a message that needs to be sent to the server. As one of the core principles of the SimplyReliable API, the Message class abstracts out the concept of a HTTP Method. For each of the HTTP Methods, the Message class has the appropriate subclass implementation defined specifically for the particular HTTP Method.
