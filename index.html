Introduction

The Simply Reliable api is a networking api specifically designed for the Android environment. The features in this api solve all the common networking problems that our faced by Android mobile apps. The following are the features of this api :

  • Make network calls to the server; that are inherently REST based
  • Ability to make synchronous and asynchronous calls to the server
  • Inherently uses JSON as the format of data exchange. Api Caller does not work at the raw json level; but uses objects for input and output
  • Success/Error and other hooks are directly built in the api, relieving the user from working at the lower HTTP level (e.g no status code checking)
  • Automatic Call retry mechanism in the event when the server is not reachable or when the client is in no/low network area. The retry mechanism is optimized for the mobile environment
  • Ability to support download of files and also support for resuming downloads
  • Call lifecycle methods that can be used to inform the status of a call to the user
  • Call (Message) Dependency. Allows the user to build a dependency graph of calls and adheres to this dependency while making the network calls
  • Integrates well with In-memory object model api's like Neebal Core Layer
