REST Semantics

The SimplyReliable API internally uses REST as the web service architectural style for making network calls to the server. The server implementation needs to be REST based, in order to be used by the api.

Http Methods

The internal REST based implementation of the api, supports the following Http Methods (Called as Verbs, in a REST terminology) :

  • GET
  • POST
  • PUT
  • DELETE
